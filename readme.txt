== Description ==

You can use this plugin in your website to add list of serial numbers and guarantee (optional) for your products in the dashboard and after that you can use a short code to display check form in a post or page to check your product if it is orginal or not using serial number or serial number and guarantee.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Genpro settings screen to configure the plugin
4. Copy the short code and paste it to use it in a post or page.
`[genpro-check-serial]`
